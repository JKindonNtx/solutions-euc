# Backup-MDTControl

Backs Up the MDT Control File.

## Syntax

```PowerShell
Backup-MDTControl
    [<CommonParameters>]
```

## Description

This function will backup the MDT control file to allow for restore once the task sequence is complete.

## Examples

### EXAMPLE 1:

```PowerShell
Backup-MDTControl 
```

Backs up the MDT control file and will return the object.
